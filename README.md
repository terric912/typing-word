## 進行方式
1. 遊戲設定：選擇題庫、音樂、音效，最後選擇遊戲時間後開始。
2. 有限時間模式：時間倒數至零結束，預設每秒鐘隨機產生一個單字。
3. 不限時模式：時間累加，自主按開關按鈕結束。
 - 起始每兩秒產生一單字，若速度追上產生單字速度，則每次減少0.1秒。
 - 單字依長度分群產生，第一群是2~6字元長、第二群是7~12字元長、第三次是超過12字元長。
